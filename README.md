Project Title

Memory Match — Focus Break Game using Python (Tkinter GUI)

Overview of the Project

Memory Match — Focus Break is an interactive brain-training game designed to improve memory and focus. The player flips hidden emoji cards to find matching pairs while the game tracks the number of moves, time taken, and matched pairs. The game stores the best performance locally and encourages users to beat their previous score. It acts as a quick and refreshing break activity to boost concentration and cognitive sharpness.

Features

16 emoji cards randomly shuffled every round

Card matching logic with visual feedback

 Live timer and move counter

 Auto-save and display of best score using JSON

 Reset / Shuffle button for replay anytime

Simple and attractive Tkinter GUI

Popup completion message after winning

Technologies / Tools Used

Category	Tools
Programming Language	Python
GUI Framework	Tkinter
Libraries	time, random, json, os
Storage	JSON File (memory_game_best.json)

Steps to Install & Run the Project

Install Python 3.8 or above
Make sure Tkinter is available (usually pre-installed with Python)
Download the project folder
Open the folder in VS Code / PyCharm / Terminal
Run the script using the command:

python memory_match.py


The game window will open — start matching the emoji pairs!

Instructions for Testing

Click on any card to reveal the emoji

Click on a second card to check for a match

If matched ➜ cards stay visible and turn green

If not matched ➜ cards flip back after a short delay

Timer and moves updated automatically

Complete all 8 pairs to win and get performance popup
