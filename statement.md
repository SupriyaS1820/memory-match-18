There is a need for a simple and refreshing brain-exercise activity that enhances focus instead of causing more stress. This project solves this issue by providing a short, mindful, and engaging memory-training game that stimulates the brain, helps improve recall ability, and boosts concentration.

Scope of the Project

This project is designed for use in short mental breaks during study or work sessions. The game is lightweight, visually appealing, and easy to play without requiring high system resources. The scope also allows future enhancements such as difficulty levels, sound effects, mobile version, themes, and leaderboard support, making it suitable for both recreational and cognitive-skill development purposes.

Target Users

The application can be used by:

Students who want to sharpen focus between study sessions

Working professionals who need a refreshing brain break

Elderly individuals for memory-improvement practice

Anyone interested in fun and mindful brain-training activities

High-Level Features

Randomized emoji cards for match-pair gameplay

Live timer and move counter to track performance

Persistent best score stored using JSON file

Simple and distraction-free GUI built with Python Tkinter

Reset / Shuffle button for replay anytime

Victory popup message displaying score summary
